Tabs

Description

A tab groups content by headings, and only allows the content of one heading to be revealed at one time, to prevent too much information from being displayed at any one time.

Note

Tabs rely on id attributes, in order to now which heading to expand and conserve state via expander classes, and also so that they can be expanded by default when a URL with an anchor is used, e.g. test.html#accordion1-item1. If multiple accordions are used on the page, ensure that these IDs are unique.
